Eke Wokocha

To run to program, simply open the main.html file. No backend is needed.

isotope.js/packery.js is running the tile filtering, sorting, and layout
tabletop.js is getting the data from the spreadsheet
angular.js is processing the data from the asynchronous call
bootstrap.css is formating the frontend
jQuery.js ties everything together
